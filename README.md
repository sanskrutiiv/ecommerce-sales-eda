# E-Commerce Sales Exploratory Data Analysis

## Project Overview
This project performs exploratory data analysis on an e-commerce sales dataset to identify trends, customer behavior, regional performance, and category-wise sales insights.

## Objectives
- Analyze sales performance across categories and regions
- Identify top-performing products and customer segments
- Understand monthly sales trends
- Explore shipping preferences and sales distribution

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Technology generated the highest sales revenue
- The West region recorded the strongest sales performance
- Standard Class was the most commonly used shipping method
- Certain sub-categories contributed significantly to total sales revenue

## Visualizations Included
- Sales by Category
- Sales by Region
- Monthly Sales Trend
- Shipping Mode Distribution
- Sales Heatmap by Region and Category

## Dataset Information
The dataset contains:
- Order details
- Customer information
- Product categories
- Regional sales data
- Shipping methods
- Sales transactions


## Conclusion
This exploratory data analysis project identified important business trends and customer purchasing patterns that can support better strategic decision-making and sales optimization.
